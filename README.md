# UBL Model Web

A website to visualize, maintain, and serve as the authoritative source for the [UBL (Universal Business Language)](https://www.oasis-open.org/committees/tc_home.php?wg_abbrev=ubl) model, replacing the current Google Sheets workflow.

## Goals

- **Visualize** the UBL data model and entity relationships
- **Edit** with change tracking and approval workflows for non-IT TC members
- **Export** GeneriCode (GC) XML files for the existing OASIS publishing pipeline
